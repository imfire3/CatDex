# CatDex

Application de collection de chats gamifiée — découvre et capture les félins de ton quartier.

## Lancer l'app

```bash
npm install
npm run dev
```

## Écrans

- **Accueil** — Hub quotidien, streak, coffre du jour, objectif personnalisable
- **Explorer** — Carte par zones, spots actifs, leurre, système de pity
- **CatDex** — Pokédex avec 18 espèces, filtres rareté, chromatiques
- **Quêtes** — Missions quotidiennes + défi hebdomadaire
- **Capture** — Rencontre sans échec possible, timing optionnel pour bonus
- **Profil** — Niveau, succès, coffres, statistiques

## Mécaniques de rétention

- Série quotidienne avec gel (style Duolingo)
- Coffre de connexion escaladant avec la streak
- Système de pity pour les chats rares (8 tentatives)
- Doublons convertis en friandises
- Bonus de retour après absence (Heure Chanceuse)
- 25 succès débloquables
- Objectif quotidien ajustable (1/3/5 chats)
